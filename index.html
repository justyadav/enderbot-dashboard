EnderBot collects and processes the following data to provide its services:

  • Server Data: Server ID, name, channel IDs, role IDs
  • User Data: User ID, username, discriminator, avatar URL
  • Message Data: Temporarily cached for edit/delete logging
  • Configuration Data: Server settings, moderation preferences

  • Service Delivery: To provide core features

  • Configuration: To store server settings
