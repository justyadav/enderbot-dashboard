🔒 Privacy Policy

How we handle and protect your data

Back to Home

1. Information We Collect

EnderBot collects and processes the following data to provide its services:

1.1 Server Data

  • Server ID: Used to identify and store settings for each server
  • Server Name: Displayed in logs and dashboards
  • Channel IDs: Used for logging and configuration
  • Role IDs: Used for auto-role functionality

1.2 User Data

  • User ID: Used to track moderation actions and warnings
  • Username: Displayed in logs and command responses
  • Discriminator: Displayed in logs and command responses
  • Avatar URL: Displayed in user info commands

1.3 Message Data

  • Message Content: Temporarily cached for edit/delete logging
  • Message IDs: Used for logging and moderation
  • Attachment URLs: Stored temporarily for logging purposes

1.4 Configuration Data

  • Server-specific settings (prefix, language, log channels)
  • Moderation settings (filters, auto-role settings)
  • Logging preferences
🔐 Privacy First: We only collect data that is essential for the Bot's functionality. We do not collect sensitive personal information.

2. How We Use Your Data

  • Service Delivery: To provide the core features of the Bot
  • Moderation: To track warnings, mutes, and other moderation actions
  • Logging: To maintain audit logs of actions taken by the Bot
  • Configuration: To store and retrieve server-specific settings
  • Improvement: To analyze usage patterns and improve the Bot

3. Data Storage & Security

  • Data is stored securely in MongoDB or in-memory (fallback)
  • We implement industry-standard security measures
  • Data is only accessible to authorized personnel
  • We do not sell or share your data with third parties

4. Data Retention

  • Server configurations are stored until the Bot is removed from the server
  • Moderation logs are stored for accountability and reference
  • Warnings are stored until manually cleared by administrators
  • Message caches are automatically cleared (max 1000 messages cached)

5. Your Rights

  • Access: You can view your server's configuration using the /config show command
  • Modification: You can modify your settings using the appropriate commands
  • Deletion: You can clear warnings and reset configurations
  • Opt-out: You can remove the Bot from your server at any time

6. Data Security

We take data security seriously and implement the following measures:

  • Encrypted connections (HTTPS for web dashboard)
  • Secure environment variables for sensitive data
  • Regular security updates and patches
  • Access controls for data management

7. Third-Party Services

  • Discord: The Bot interacts with Discord's API
  • MongoDB: Optional database for storing data (if enabled)
  • FastAPI: Used for the web dashboard (if deployed)

These services have their own privacy policies that you should review.

8. Children's Privacy

EnderBot is not directed at children under 13 years of age. We do not knowingly collect personal information from children. If you believe we have collected such information, please contact us.

9. Changes to Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new policy on our website. You are advised to review this policy periodically.

10. Data Removal

If you want your data removed from EnderBot:

  • Server Admins: Remove the Bot from your server, and all server data will be deleted
  • Individual Users: Contact a server admin to clear your warnings
  • All Users: Contact us via the support server for data removal requests
📧 Contact: For privacy concerns, please reach out through our Support Server.

11. International Data Transfers

Your data may be transferred to and maintained on servers located outside of your country of residence. By using EnderBot, you consent to this transfer.

12. Cookies

EnderBot does not use cookies directly. However, the web dashboard may use session data for authentication purposes.
